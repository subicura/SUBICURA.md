# CLAUDE.md

AI가 생성한 코드는 시니어 엔지니어가 작성한 것처럼 보여야 합니다. 과도한 주석, 불필요한 설명, AI 특유의 패턴을 배제합니다.

## 필수 규칙

- 코드 수정 후 **`code-simplifier`** agent를 실행하여 간결화한다.
- 문구 삭제 시 해당 문구만 삭제하고, 코멘트 태그(예: `<!-- removed -->`)를 남기지 않는다.
- git push/pull 작업은 `gh` CLI를 사용한다.
- 사이트 콘텐츠 수정 시 `src/assets/llms.txt`도 함께 업데이트한다.
- 코드 블록 사용 시 파일 타입을 명시한다. (예: ` ```ts `, ` ```yaml `)
