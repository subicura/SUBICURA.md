# SUBICURA.md

[subicura.md](https://subicura.md) — human agent configuration file

## Stack

- [Eleventy (11ty)](https://www.11ty.dev/) — static site generator
- Markdown + Nunjucks layout
- PostHog analytics

## Development

```bash
npm install
npm run dev    # 로컬 서버 (hot reload)
npm run build  # _site/ 디렉토리에 정적 파일 생성
```
